## 基础语法规范

### 标题层级演示
````markdown
# H1主标题
## H2副标题
### H3节标题
#### H4子标题
##### H5特殊注释标题
###### H6超小标题（部分工具会忽略渲染）
````
> 我还整理了一张表格，如下：

| Markdown 语法 | 名称       | HTML 标签   | 典型用途                     |
|--------------|-----------|------------|-----------------------------|
| `#`          | 一级标题   | `<h1>`     | 文档主标题（通常唯一）         |
| `##`         | 二级标题   | `<h2>`     | 章节标题                     |
| `###`        | 三级标题   | `<h3>`     | 子章节标题                   |
| `####`       | 四级标题   | `<h4>`     | 子段落标题                   |
| `#####`      | 五级标题   | `<h5>`     | 特殊注释标题                 |
| `######`     | 六级标题   | `<h6>`     | 极小文字标题                 |

##### 效果展示：
# H1主标题
## H2副标题
### H3节标题
#### H4子标题
##### H5特殊注释标题
###### H6超小标题（部分工具会忽略渲染，我用的这个主题似乎将它化为了**引用**的样式了）
Tips：不同博客主题的标题可能字体会不一样（渲染不同）

## 高级语法应用

### 代码块
````markdown

```
````
##### 示例代码：
````markdown
```python

```
````

print("Hello, world!")
