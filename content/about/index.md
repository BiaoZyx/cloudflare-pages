# 关于此站点

# 为何建立
2024,我本想自己搭建一个网站，但奈何当时能力不够，于是我从2024年12月开始在博客园写博文，一切尚好。我还找到了一个花哨、带播放器的配置。

但突然，在2025某天，我在新装好的Debian打开自己的博客配置GNOME，却发现居然有广告在底部！忍无可忍，于是默默期待某天能够真的搭建出自己的网站……

今年，2026,我终于圆梦。我的博文能够无拘无束地被展现！

## 简介
一个由 Alpine Linux 爱好者维护的小小技术站点。

这里的核心叙事，围绕着一个虚构的“营地”世界展开：

-   **用户 (User): Traveller**，也就是我。
-   **工具 (Tool): Photographer**，用来记录系统状态的快照。
-   **快照集合 (Snapshot Collection): Campsite**，所有系统快照的“家”。
-   **单个快照 (Individual Snapshot): Tent**，一个可回滚的系统状态。

**关于内容**

这里记录的是我的系统探索实践，主要关注：

- Linux 相关教程以及笔记
- 我的一些开发板折腾随笔
- 杂七杂八需要共享的文章

**关于我**

一个普通的 Linux 玩家，享受折腾系统带来的乐趣和掌控感，最喜爱Alpine Linux的 *S. S. S.* 和 *K. I. S. S.*。

**用一句话总结这个营地：**

> “系统是工具，数据是核心，乐趣在过程，而不在于复杂度。”
